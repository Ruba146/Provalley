# Provalley – Java OOP Project

This repository contains the full Object-Oriented Programming project I developed, divided into two main parts:

---

##  1) ProjectCode
This folder contains the main Java project, including:

- `build/`
- `classes/`
- `images/`
- `nbproject/`
- GUI resources
- Project source code

###  Features:
- Multi-window Java application  
- Image-based GUI design  
- OOP principles such as classes, encapsulation, and object management  
- NetBeans project structure  

---

##  2) TimerCode
This folder contains a simple standalone Java Timer project.

###  Features:
- TimerAppo.java  
- Simple GUI  
- Countdown timer functionality  

---

##  Technologies Used
- Java  
- NetBeans IDE  
- Object-Oriented Programming (OOP)

---

##  How to Run the Projects

###  Running ProjectCode:
1. Open **NetBeans**
2. Go to **File → Open Project**
3. Select the folder **ProjectCode**
4. Click **Run Project**

###  Running TimerCode:
1. Open `TimerAppo.java` in NetBeans
2. Right click → **Run File**
